# Search Engine Optimization (SEO) Skill

## Overview
SEO is the practice of orienting your website to rank higher on a search engine results page (SERP) so that you receive more traffic. The focus here is on **Technical SEO** and **On-Page Optimization**.

## Core Concepts

### 1. Technical SEO
Technical SEO refers to website and server optimizations that help search engine spiders crawl and index your site more effectively.
* **Crawlability:** Ensuring `robots.txt` is configured correctly.
* **Indexability:** Using `canonical` tags to prevent duplicate content.
* **Site Architecture:** Clean URL structures and logical internal linking.
* **HTTPS:** Google treats security as a ranking signal.

### 2. Core Web Vitals (Performance)
Google measures user experience through:
* **LCP (Largest Contentful Paint):** Loading performance.
* **FID (First Input Delay):** Interactivity.
* **CLS (Cumulative Layout Shift):** Visual stability.

### 3. Semantic HTML & Metadata
* **Title Tags & Meta Descriptions:** Critical for Click-Through Rate (CTR).
* **Header Hierarchy:** Proper use of `<h1>` through `<h6>`.
* **Alt Text:** Optimization for image search and accessibility.
* **Structured Data (JSON-LD):** Helping Google understand the context of your data.

## Checklist for Ranking
- [ ] Mobile-friendliness test passed.
- [ ] Page load speed under 2.5 seconds.
- [ ] No broken links (404 errors).
- [ ] Sitemap.xml submitted to Google Search Console.
- [ ] Optimized `meta title` (under 60 chars) and `description` (under 160 chars).

## Tools
* **Google Search Console:** Monitoring index status.
* **Google Analytics:** Tracking user behavior.
* **Lighthouse:** Auditing performance and SEO.
* **Ahrefs / SEMrush:** Keyword research and backlink analysis.
